+ Follow python cubes custom installation and install both requirements and optional requirements
+ Install XAMPP: https://soft98.ir/internet/webmaster-tools/13861-xampp.html
+ Install visual studio build tools 14: http://go.microsoft.com/fwlink/?LinkId=691126&fixForIE=.exe
+ pip install mysql-connector mysqlclient
+ Start mysql from XAMPP
+ Start apache from XAMPP
+ Open your browser and go to localhost/phpmyadmin
+ Login with root and leave password empty
+ Create database olapdb
+ Import irbd_balance.sql to olapdb
+ Run examples
+ Note: data.csv and prepare_data.py are not needed for this example. They just show how to insert csv data into mysql.
